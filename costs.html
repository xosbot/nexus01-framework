Tech Stack & Cost Analysis

Every choice justified. Every dollar accounted for.

Technology Choices

Every tool in the stack was chosen for a specific reason. No bloat, no hype.

Language
Python 3.11+
Async-first, type-hinted, massive ecosystem. The lingua franca of AI.
Why: Ollama SDK, ChromaDB, and all ML tools are Python-native.
LLM Runtime
Ollama
Local LLM inference. Zero API costs. Full model control. One command setup.
Why: Eliminates API dependency. Your data never leaves your machine.
Database
SQLite
Embedded, zero-config, ACID-compliant. The most deployed database in the world.
Why: Single file, no server process, handles 99% of use cases.
Vector Store
ChromaDB
Python-native vector database. Embedded mode for local, client-server for scaling.
Why: Simplest API. No separate server. Perfect for local-first.
HTTP Client
httpx
Async HTTP client with connection pooling. Drop-in replacement for requests.
Why: Native async support for concurrent web scraping.
Scraping
BeautifulSoup4
Battle-tested HTML parser. Forgiving, fast, and handles messy real-world HTML.
Why: Pairs with httpx for lightweight scraping without browser overhead.
Embeddings
sentence-transformers
Local embedding generation. all-MiniLM-L6-v2 for speed, bge for quality.
Why: Zero API cost embeddings. 384-768 dimensions. Runs on CPU.
Interface
Telegram Bot
Conversational interface. Rich messages, inline keyboards, file sharing.
Why: Free, reliable, works on every device. Best bot platform.
Containerization
Docker
Isolated environments. Reproducible builds. Easy deployment.
Why: One-command deployment. Consistent across dev and prod.
Browser Automation
Playwright
Cross-browser automation. Headless Chrome, Firefox, WebKit.
Why: Modern, fast, MCP-compatible. Better than Selenium.
CLI Framework
Rich
Beautiful terminal output. Tables, progress bars, syntax highlighting.
Why: Makes CLI output professional without extra effort.
Task Scheduling
APScheduler
Cron-like scheduling in Python. Interval, cron, and date triggers.
Why: Built-in persistence. Survives restarts. No external daemon.

The Real Cost of Cloud AI

💰

API Token Costs

GPT-4o: $2.50/1M input tokens. A busy agent can burn $50-100/month in API calls alone.

🔒

Data Privacy Risk

Every prompt, every document, every code snippet sent to third-party servers. Compliance nightmare.

🚨

Vendor Lock-in

API changes, price hikes, service deprecation. Your entire workflow depends on someone else's uptime.

⏰

Latency & Rate Limits

Network round-trips add 200-500ms per call. Rate limits kill batch processing workflows.

📄

Compliance Costs

SOC2, GDPR, HIPAA audits when data flows through external APIs. Thousands in legal fees.

📈

Scaling Surprises

What starts at $20/month becomes $200/month as usage grows. No cost predictability.

Cost FAQ

Can I really start for $0?

Yes. Ollama runs local LLMs, SQLite stores data, ChromaDB handles vectors, Telegram is free. All on your existing machine. No credit card needed.

When should I upgrade to production?

When you need 24/7 uptime, remote access, or higher throughput. A $5/month VPS handles most production workloads with room to grow.

What about GPU costs?

Not required. Ollama runs on CPU with 8B parameter models. GPU only needed for 70B+ models or high-throughput scenarios.

How does this compare to SaaS alternatives?

AutoGPT Pro: $40/mo. ChatGPT Team: $25/user/mo. Enterprise AI platforms: $500+/mo. NEXUS-01: $0–$42/mo with full data control.
