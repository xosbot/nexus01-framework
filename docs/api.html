API Reference

Core modules, classes, and interfaces in NEXUS-01.

Message Bus

Inter-agent communication via publish/subscribe or request/response.

from core.bus import Message, MessageBus

# Create a message
msg = Message(
    sender="orchestrator",
    recipient="osint",
    type="task",
    payload={"query": "AI agent frameworks 2026"}
)

# Request/response (waits for reply)
reply = await bus.request(msg, timeout=30.0)

# Fire and forget
await bus.publish(msg)

# Subscribe to messages
bus.subscribe("osint", handler_fn)

Cold Mode Gate

5-step risk verification before any critical action.

from core.cold_mode import ColdMode

cold_mode = ColdMode(enabled=True)

context = ColdMode.build_context(
    action="execute_command",
    permission="EXECUTE",
    confidence=0.85,
    fallback_script="echo rollback",
    numeric_values=[100, 200]
)

if cold_mode.should_block(context):
    reasons = cold_mode.get_failure_reasons(context)
    print(f"Blocked: {reasons}")
CheckDescription
Data VerificationIs source reliable?
Parameter ValidationWithin normal ranges?
Confidence AssessmentModel confidence ≥ 0.75?
Risk EvaluationWhat happens if wrong?
Fallback DeterminationIs there a safe default?

LLM Router

Multi-provider LLM with tiered routing and circuit breaker.

from core.llm_router import LLMRouter

router = LLMRouter(config_path="core/llm_config.yaml")

# Simple chat
response = await router.chat(messages, tier="standard")

# With tool calling
response = await router.complete_with_tools(
    messages, tools=tools_schema, tier="standard"
)

# Streaming
async for token in router.stream(messages):
    print(token, end="", flush=True)

# Provider status
status = router.provider_status()

RAG Pipeline

Document ingestion and semantic search via ChromaDB.

from core.rag import RAGStore

# Search

Memory

SQLite + ChromaDB persistence for conversations and knowledge.

from core.memory import Memory

memory = Memory("./data/nexus.db", "./data/chromadb")

# Save conversation
memory.save_conversation("osint", "user", "research AI agents")
memory.save_conversation("osint", "assistant", "Here are the findings...")

# Get context for agent
history = memory.get_context("osint", last_n=6, session_id="chat-123")

# Save knowledge
memory.save_knowledge("ai_agents_2026", "Research data...", {"agent": "osint"})

Docker Sandbox

Isolated container execution for the Executor agent.

from core.sandbox import DockerSandbox, SandboxConfig

config = SandboxConfig(
    image="python:3.12-slim",
    cpu_limit=0.5,        # 0.5 CPU cores
    memory_limit="256m",  # 256MB RAM
    timeout_seconds=30,   # 30s hard limit
    network_disabled=True,
    read_only_root=True,
)

sandbox = DockerSandbox(config)

# Execute Python code
result = await sandbox.execute("print('hello')", language="python")

# Run a shell command
result = await sandbox.run_command("ls -la /tmp")

print(result.stdout, result.exit_code, result.duration_ms)
Security: Sandbox containers have no network access, read-only root filesystem, and strict CPU/memory limits. The container is auto-removed after execution.

Telegram HITL

Human-in-the-loop approval management for destructive operations.

from gateway.hitl import ApprovalManager, ApprovalRequest

manager = ApprovalManager()

# Create approval request
req = manager.create(
    action="exec",
    description="Delete production database",
    command="rm -rf /data/prod/*",
    requester="user:123",
    channel="telegram",
    session_id="chat-456"
)

# Check status
pending = manager.get(req.id)

# Approve or deny
manager.approve(req.id)
# or
manager.deny(req.id)

# Cleanup expired
expired_count = manager.cleanup_expired()
