Quickstart

Get NEXUS-01 running on your machine in under 5 minutes.

Prerequisites

  • Python 3.11 or higher
  • Ollama installed and running
  • Docker (optional, for sandboxed executor)

Installation

1

Clone the repository

git clone https://github.com/xosbot/nexus01-framework.git
cd nexus01-framework
2

Install dependencies

pip install -r requirements.txt
3

Pull an LLM model

ollama pull llama3.1
4

Run NEXUS-01

python main.py
That's it! You should see the NEXUS-01 prompt. Type help to see available commands.

Configuration

NEXUS-01 is configured via environment variables or a config.yaml file.

Minimal config.yaml

# config.yaml
ollama_url: "http://localhost:11434"
ollama_model: "llama3.1"
cold_mode_enabled: true
require_approval_for_exec: true
enable_web_ui: true
api_port: 8765

Environment Variables

# Core
export OLLAMA_URL="http://localhost:11434"
export OLLAMA_MODEL="llama3.1"
export COLD_MODE_ENABLED="true"

# Telegram (Phase 3)
export TELEGRAM_TOKEN="your-bot-token"
export TELEGRAM_ALLOWED_USERS="your-telegram-id"

# Cloud LLM (optional)
export GEMINI_API_KEY=""
export GROQ_API_KEY=""
export OPENAI_API_KEY=""
export ANTHROPIC_API_KEY=""

# Phase 3
export EXECUTOR_SANDBOX_ENABLED="true"
export STRUCTURED_LOG_JSON="false"
export BUS_BACKEND="local"  # or "redis"
export REDIS_URL="redis://localhost:6379"

Telegram Setup

To control NEXUS-01 via Telegram:

  1. Message @BotFather on Telegram
  2. Send /newbot and follow the prompts
  3. Copy the bot token
  4. Set TELEGRAM_TOKEN environment variable
  5. Restart NEXUS-01
export TELEGRAM_TOKEN="7123456789:AAH..."
python main.py
Security: Set TELEGRAM_ALLOWED_USERS to your Telegram user ID to restrict access. Send /start to your bot to get your user ID.
