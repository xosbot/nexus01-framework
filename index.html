Architecture Overview

Six layers of intelligence. Click each layer to explore.

06
🖥️

Interfaces

Telegram Bot, Rich CLI, Web UI
▸
The user-facing layer. Communicates via Telegram bot for conversational access, a Rich-powered CLI for terminal workflows, and a future web dashboard for visual monitoring and control.
05
🎯

Orchestration

Task routing, scheduling, priority queues
▸
The brain's dispatcher. Routes tasks to the right agent, manages scheduling, handles priority queues, and coordinates multi-step workflows across the agent team.
04
🤖

Agents

OSINT, Execution, Analytics
▸
Specialized autonomous agents. The OSINT Agent gathers intelligence, the Execution Agent performs actions, and the Analytics Agent processes and summarizes data. Each operates independently within the orchestration framework.
03
📡

Communication

Inter-agent messaging, XClaw protocol
▸
Agents talk to each other via structured messaging. The XClaw-x-DClaw protocol enables inter-agent communication, allowing agents to collaborate on complex multi-step tasks autonomously.
02
🗄️

Knowledge

SQLite, ChromaDB, embeddings
▸
The memory layer. SQLite stores structured data and state. ChromaDB provides vector storage for semantic search. Sentence-transformers generate embeddings for context-aware retrieval and reasoning.
01
🔮

LLMs

Ollama, local models
▸
The foundation. Ollama runs local LLMs (Llama 3, Mistral, CodeLlama) entirely on your hardware. Zero API costs, zero data leaving your machine, full control over model selection and parameters.
00
🏗️

Infrastructure

VPS, Docker, monitoring
▸
The bare metal. A $5/month VPS running Ubuntu with Docker containers, system monitoring, log aggregation, and automated backups. The minimal infrastructure that makes everything else possible.
