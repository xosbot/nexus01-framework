System Architecture

Every component is modular. Swap any part without breaking the whole.

Built for Real Work

Not another chat wrapper. Production-grade agent infrastructure.

🐳

Docker Sandbox

Executor runs in isolated containers. CPU limits, memory caps, network disabled, read-only filesystem. 30s timeout.

🛡️

Cold Mode Protocol

5-step verification gate. Data validation, confidence scoring, risk evaluation, fallback determination. Blocks before harm.

📱

Telegram HITL

Inline keyboard approval for destructive ops. Approve from your phone. 5-min expiry. Full audit trail.

🔍

Firecrawl OSINT

Structured intelligence reports. Firecrawl scraping with DuckDuckGo fallback. LLM-synthesized analysis.

⚡

Cloud LLM Router

Ollama → Gemini → Groq → Claude. Tiered routing, circuit breaker, streaming, per-request cost tracking.

🧠

ReAct Agent Loop

Reason + Act with parallel tool execution. RAG context injection. Up to 15 iterations per task.
