Common Questions

Everything you need to know about implementing NEXUS-01 use cases.

Can I run multiple use cases simultaneously?

Yes. NEXUS-01's orchestrator manages concurrent agent execution. Each use case runs as an independent skill, and the task queue handles prioritization and resource allocation.

Do I need cloud APIs for any use case?

No. Every use case works with Ollama for LLM inference. Cloud APIs (GPT-4o, Claude) are optional upgrades for higher quality, but the $0 MVP tier covers all use cases.

How do I add custom use cases?

Create a new SKILL.md file in the skills directory. Define triggers, instructions, and resources. The orchestrator automatically discovers and routes to your new skill.

What hardware do I need?

Minimum: 4GB RAM, 2 CPU cores for Ollama. Recommended: 8GB+ RAM, 4+ cores for smooth multi-agent operation. No GPU required for 8B parameter models.

Can I scale to production workloads?

Absolutely. Start local for development, deploy to a $5 VPS for production. Scale to GPU instances for high-throughput scenarios. The architecture supports horizontal scaling.

How does Cold Mode work with use cases?

Cold Mode activates for high-stakes decisions. When enabled, agents verify data through 3+ sources, set confidence thresholds, and prepare fallback plans before executing.
